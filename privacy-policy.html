Privacy Policy for Upload Bot

Last updated: May 30, 2026

1. Information We Collect

Upload Bot does not collect, store, or share any personal information from users. The application accesses TikTok account data solely for the purpose of uploading videos on behalf of the authorized account owner.

2. Use of TikTok API

This application uses TikTok's Content Posting API to automate video uploads. All API credentials are stored securely on the user's local device using the operating system's keychain. No credentials are transmitted to third-party servers.

3. Data Storage

All data (video files, metadata, credentials) is stored locally on the user's device. We do not operate any servers or databases.

4. Data Sharing

We do not sell, trade, or transfer any information to third parties.
